# DashboardLab - Week 4 Learning Project

## Overview

This is a learning project for **Week 4: Frontend Integration (HTML, CSS, JavaScript)**.

The goal is to practice building a simple dashboard to visualize data from .NET backend services.

## Learning Objectives

- DOM manipulation and event handling
- Fetch API with async/await
- Client-side data validation
- Dynamic table rendering
- Regex validation in JavaScript
- Semantic HTML5
- Responsive CSS (Flexbox, Grid)
- Error handling and user feedback

## Project Flow

```
1. Fetch scraping results from .NET API
2. Fetch OCR results from .NET API
3. Render interactive tables
4. Add filtering and sorting
5. Display validation status
6. Handle errors gracefully
```

## Key Concepts Practiced

- Fetch API and promise handling
- DOM event listeners and manipulation
- Dynamic HTML generation
- CSS Grid and Flexbox layouts
- Validation message display
- Responsive design
- Error state UI

## Deliverables

- Functional dashboard HTML
- CSS styling (responsive)
- JavaScript interactivity
- API integration
- Error handling UI
