# MiniETL

Learning project for .NET and C# fundamentals.

## Overview

This project practices foundational concepts by building a simple ETL (Extract-Transform-Load) pipeline.

## Learning Objectives

- .NET 8 Console project structure
- HttpClient + async/await patterns
- JSON serialization with System.Text.Json
- File I/O and streams
- Interfaces and dependency injection basics
- Error handling and logging setup
- Data validation patterns

## Project Flow

```
1. Extract: Download JSON from public API
2. Transform: Parse and validate data
3. Load: Export to CSV file
```

## Key Concepts Practiced

- Async/await with HttpClient
- Generic service interfaces (IDataService)
- Configuration management
- Logging basics
- CSV export

## Deliverables

- Functional ETL pipeline
- Basic error handling
- Input validation
- CSV output
