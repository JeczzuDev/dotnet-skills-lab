# WebScraperLab

Learning project for web scraping techniques.

## Overview

This project practices web scraping by extracting data from a real website using HtmlAgilityPack and advanced Regex patterns.

## Learning Objectives

- HtmlAgilityPack for HTML parsing
- XPath and CSS selectors
- Advanced Regex patterns (named groups, lookaheads)
- Data validation and cleansing
- Pagination handling
- CSV export

## Project Flow

```
1. Fetch HTML from website
2. Parse DOM with HtmlAgilityPack
3. Extract data using XPath/CSS selectors
4. Validate with Regex
5. Export to CSV
```

## Deliverables

- Functional web scraper
- Data validation
- CSV export
- Error handling
