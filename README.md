# .NET Skills Lab

## Overview

A comprehensive, **self-paced learning program** designed to develop production-ready skills in:

- **C# & .NET** (modern, enterprise patterns)
- **Web Scraping** (advanced, real-world scenarios)
- **OCR & Image Processing** (computer vision, data extraction)
- **Frontend Integration** (HTML, CSS, JavaScript)
- **Testing & CI/CD** (automation, quality assurance)
- **Technical English** (documentation, communication)

This repository contains both **learning projects** and **portfolio projects** that build upon each other to create production-grade applications.

## Repository Structure

```
dotnet-skills-lab/
│
├── Learning/                          # Educational projects

│   ├── WebScraperLab/

│   ├── DashboardLab/
│   ├── TestingCIPipeline/
│   └── EnterprisePatternsRefactor/
│
├── Portfolio/                         # Production-ready projects
│   ├── WebScraper-Advanced/
│   ├── OCRPipeline-Production/
│   └── AutomationOrchestrator/
│
└── README.md                          # This file
```

Each project folder contains:
- `src/` - Source code
- `tests/` - Unit and integration tests
- `.github/workflows/` - CI/CD configuration
- `README.md` - Project-specific documentation
- `docs/` - Architecture, API, troubleshooting guides


###  Portfolio Phase

Three production-grade projects integrating all learning:

| Project | Purpose | Technologies |
|---------|---------|--------------|
| **WebScraper-Advanced** | Real-world scraping system | HtmlAgilityPack, Regex, validation, parallel processing |

## Quick Start

### Prerequisites
- .NET 8.0+
- Git
- Visual Studio Code (or preferred IDE)

### Get Started

```bash
# Clone repository
git clone https://github.com/JeczzuDev/dotnet-skills-lab.git
cd dotnet-skills-lab

# Start with Lesson 1
cd Learning/MiniETL
dotnet restore
dotnet run

```
